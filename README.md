# Buy-Anabolics

### Project initialization

For cloning project:

```bash

git clone https://github.com/kemaldeveloper/buy-anabolics.git

```

For install all dependencies:

```bash

$ yarn install

```

For start dev server:

```bash

$ yarn dev

```

For build project:

```bash

$ yarn build

```

### Dependencies
