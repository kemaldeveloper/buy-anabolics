# Buy-Anabolics

### Project initialization

For cloning project:

```bash

git clone https://github.com/kemaldeveloper/buy-anabolics.git

```

For install all dependencies:

```

$ yarn install

```

For start dev server:

```

$ yarn dev


```

For build project:

```

$ yarn build

```

### Dependencies

- @popperjs/core 2.11.8
- bootstrap 5.3.5
- choices.js 11.1.0
- jquery 3.7.1
- nouislider 15.8.1
- swiper 11.2.6
